Welcome to the Utility Base!

UtilityBase is a comprehensive toolkit designed to simplify your everyday tasks. Here's a quick guide to help you get started:



· Base64 Encoder/Decoder

You can either enter text or upload files for Base64 encoding or decoding.

Encoder only accepts UTF-8 format, all other format characters may cause errors or wrong encoding.

Please ensure that files are no larger than 3GB.

Keep in mind that higher file sizes may cause the site to run slower due to its Python-based infrastructure.

· JSON Minifier/Beautifier

You can input JSON text directly or upload a JSON file to minify or beautify.

Similar to the Base64 tool, ensure that JSON files do not exceed 3GB in size for optimal performance.

· URL Encoder/Decoder

Enter a valid URL for encoding or decoding.

Make sure the URL starts with "http://".

· QR Code Generator

Generate QR codes from text or links.

Simply input the desired text or URL to get your QR code.

· URL Shortener

Enter a valid URL for encoding or decoding.

Make sure the URL starts with "http://".

If you encounter any issues or have questions, don't hesitate to reach out to us. Enjoy using UtilityBase!
